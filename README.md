## 说明
一个webpack的配置demo，供大家参考。会逐步更新。
推荐使用 cnpm，速度有保证。
## 常用命令
### 启动(有mock数据)
```bash
npm run start
```
### 启动(没有mock数据)
```bash
npm run start:no-mock
```
### 打包
```bash
npm run build
```
### 测试
```bash
npm run test
```
### e2e测试
```bash
# 务必先npm run start启动，再运行下面的测试
npm run test:e2e
```
### 代码检查
```bash
npm run check
```
### 打包模块分析
```bash
npm run analyze
```
